# Routine & Streak Tracker

A routines and streak tracker built using HTML, CSS, and vanilla JavaScript, integrated with Supabase for cloud data synchronization.

## Features
- Daily checklist with progress visualization.
- Weekly, calendar, and habit management views.
- Light and dark theme modes.
- Guest demo sandbox mode.
- Local and Supabase data synchronization.

## How to Run
Open `index.html` in a web browser.
